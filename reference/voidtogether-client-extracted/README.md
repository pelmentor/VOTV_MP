# VoidTogether SANDBOX CONCEPT DEMO
***Voices of the Void, Together with Friends.***

The First Multiplayer Mod, based around Websockets and a NodeJS Server.

[Click Here to Start Setting Up Your Own Server](https://github.com/VoidTogether/VoidTogether-Server)

## Important Controls
- On the main menu, go to Configure Mods and you can change your Username from "KelNumber1234"

- Right-Alt Key > Opens VoidTogether Ingame Menu for Swapping Models, Options or Moderation Stuff

- Slash ( / ) Key > Types in Chat

## THIS IS A SANDBOX CONCEPT DEMO
If you host a custom server, you can choose to plaly on the default Voices of the Void map in the `serverconf.yml`
though it is highly advised you do not

The following things are currently not supported by VoidTogether in this Demo State:
- Doors
- NPCs (Kerfur, Furfur, Arirals, etc)
- Story Event Syncing
- Ambient Event Syncing
- Prop Functions (Using, Toggling, etc are not supported yet for various reasons)
- Scanning Signals
- Using the Computer, Emails, and Delivery Drone
- Daily Tasks
- Certain Ambient Weather
- Saving
- Custom Content (TV, Radio, Pictures, 3D Printer)

## Current Features:
- Online Multiplayer Sandbox Gamemode
- Multiple Servers, including Official Server(s), and a Custom Server Browser Menu
- Custom Player Models from VotV and other EternityDev Games
- Chat and Commands System, based around Permissions
- Administration Commands and Tools
- Player and Limited Prop Syncing
- Custom Server Support (Plugin Support SoonTM)
