# **VoidMod Loader for VotV**
VoidMod is a `Unreal Engine 4.27.2` Blueprint Framework/API and Handler for Modding *Voices of the Void*.
Completely Rewritten as of 6/11/2024!

VoidMod Supports:
- Custom Events (Story and Random)
- Custom Props (Thanks to Fusion, supports fixes for custom prop content)
- Time Control/Conditions
- Player Stats Control
- Custom UI
- Saving/Loading Variables per save
- Custom Satellite/Root Console Commands (Custom sv. commands)
- Mod Dependencies Fetching

## Installing Mods
If you are not using Thunderstore/R2ModManager to install your mods, you may follow the guides left on the [Voices of the Void Modding Wiki](https://votvmodding.github.io/docs/welcome).

R2ModManager/Thunderstore should automatically install VoidMod compatible mods for you just by using the Download Button.
